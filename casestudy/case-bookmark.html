Case Study : Bookmark Application

Are you one of those people who has hundred of bookmarks across multiple browsers? Then this web based application is for you!

The Bookmark Management System allows users to add, store, update and delete entries of content through the web applications form based interaction.

The JavaScript web based application utilises a browsers local storage & can house multiple URL's & accomodating url name into a neatly compiled list


Bookmark Management Application

Overview


A web based application utilising JavaScript's reactive and interactive features. The application takes a users input from the two search parameters provided in the form of an input element. After completing the form and clicking "save", a JavaScript element will be created under the "Saved Bookmarks" heading. This element houses data taken from the browsers local Storage and displays it to a user with a visit url button and a delete button.

Development


Developed this web application to better understand creating, appending & deleting content using JavaScript & the browsers local Storage. This project gave a great insight to multiple new JavaScript features including JSON's stringify argument that turns JavaScript objects into a strings for communication between web servers. JSON's parse function does the opposite and outputs JSON string data back to a JavaScript object.

Testing & Diagnostics


Testing takes place using multiple browsers to test compatibility. Lauching on a local machine web server (local host) for development and testing purposes. This ensures a responsive design and layout across multiple browsers and devices.
