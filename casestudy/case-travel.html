Case Study : Travel Brochure

World adventurer, traveling tourist, family vacation? At NorthWest we offer a wide range of travel packages, accommodation deals, sightseeing adventures and much more!

The Travel Brochure produces a clear and informative approach to searching for the latest travel options, deals, locations, ancient sites & popular tourist attractions.


Travel Brochure Website

Overview


A website developed based on my love for travelling, exploring culture and ancient temples, discovering quaint ski towns, and lazy relaxation beach getaways. Each page give the user a small amount of content & informative navigation to explore the website with ease of use.


XD Concept


Took inspiration from two different web layouts on https://awwwards.com and combined to create an amazingly presented travel brochure. Below is a screenshot from the XD design.


Design


Float layouts for the navigation panel, the main company heading & the main navigation. Each page displays a unique color theme depending on the type of page the user is viewing. The website features responsive images, a responsive flex layout above 960px, and multi colored themed backgrounds.

Development


Having a booking app that is responsive across multiple devices was a top priority. In creating a multi-page web layout with each page representing a different theme & color scheme, took a great deal of problem solving, create a better understanding of naming conventions with regards to HTML & CSS.

Testing & Diagnostics


Testing takes place using multiple browsers to test compatibility. Lauching on a local machine web server (local host) for development and testing purposes. Creating a live site on github to test online features & issues. Multiple devices are also used for testing to ensure there are specific media query breakpoints in place. This ensures a responsive design and layout across multiple browsers and devices.
